A plugin that allows character number limit in Title


== Description ==


* Limit the number of characters in Title.

* Very useful to limit characters in the title and may also set or not inserting an ellipsis (...) at the end of the title.

* This plugin only limits the number of characters and stops the process the current long title of the publication.


== Installation ==


* Upload the plugin to the wp-content/plugins folder via FTP or use the default installer Plugins -> Add New and then click the Send tab.

* Activate the plugin.

== Frequently Asked Questions ==


= So how do I limit characters in the title? =


* To limit characters in the title of your post or page, type in the "Number of characters" the value of your choice.


* In option "ellipsis (...)", you can choose whether or not the ellipsis at the end of the title.
